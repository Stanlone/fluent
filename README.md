# project-template

Template repo for new projects. Contains:

- `BOOTSTRAP.md` — step-by-step guide for starting a new project
- `WORKFLOW.md` — process rules for the entire project lifecycle
- `.claude/CLAUDE.md` — technical constants template (filled in during bootstrap)

## Usage

```bash
gh repo create [project-name] --template Stanlone/project-template --private --clone
```

See BOOTSTRAP.md for full instructions.
